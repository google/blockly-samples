Step Execution with Python Interpreter

This is a demo of executing code step-by-step with a sandboxed Python interpreter.

The generator's python.pythonGenerator.STATEMENT_PREFIX is assigned 'highlightBlock(%1)\nawait breakpoint()\n', where %1 is the block ID. The call to highlightBlock() will highlight the identified block and the call to await breakpoint() will pause the execution.

Each press of the "Step Python" button will run the interpreter one step. That is, until highlightBlock() has highlighted the block that will be executed on the next step.
